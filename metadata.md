# 내부 사용을 위한 메타데이터

[!DNL metadata].md 파일에는 리포지토리의 사용 안내서 [!DNL TOC.md] 파일에 전달되는 리포지토리 수준 메타데이터가 포함되어 있습니다. 사용 안내서의 metadata.md 콘텐츠를 변경하려면 TOC.md 파일에서 이 작업을 수행합니다.

**솔루션**, **제품** 및 **유형**&#x200B;에 대한 올바른 값 목록은 [메타데이터](https://experienceleague.adobe.com/docs/authoring-guide-exl/using/editing/user-guide-setup/metadata.html?lang=en)를 참조하십시오.
