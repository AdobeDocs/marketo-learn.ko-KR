# Dynamic Chat 튜토리얼

![](assets/dynamic-chat-header.png)

[!UICONTROL Dynamic Chat] 튜토리얼 사이트에 오신 것을 환영합니다. 다음 튜토리얼과 설명서를 통해 [!UICONTROL Dynamic Chat]를 사용하여 맞춤형 대화를 만드는 방법에 대한 이해를 높이십시오. 다음 주제를 통해 시작할 수 있습니다.

* 개요
   * **[빠른 제품 둘러보기 [2023년]](product-tour.md)**
     *2023년에 출시된 모든 새로운 기능에 대한 개요를 확인하십시오.*
   * **[빠른 제품 둘러보기 [2022년]](product-tour.md)**
     *전체 제품에 대한 개요를 확인하고 [!UICONTROL Dynamic Chat]을 활용하는 방법을 알아봅니다.*
   * **[15분 만에 시작하기](go-live-in-15-minutes.md)**
     *단 15분 안에 [!UICONTROL Dynamic Chat]을 사용해 첫 대화를 진행하는 데 도움이 되는 단계별 안내서입니다.*
* 설정
   * **[챗봇 설정 및 설치](setup.md)**
     *[!UICONTROL Dynamic Chat] 구독을 설정하는 단계를 알아봅니다.*
   * **[[!UICONTROL Dynamic Chat]](marketo-integration.md)**과 Marketo 통합 활성화
     *Marketo에서 제공하는 리드 정보를 활용하여 웹 방문자에게 맞춤형 대화를 유도하고 개인화된 서비스를 제공합니다.*
   * **[[!UICONTROL Dynamic Chat]](user-management.md)** 사용자 추가 및 관리
     *[!UICONTROL Dynamic Chat]에서 사용자를 추가하거나 관리하는 방법에 대해 알아봅니다.*
   * **[에이전트 관리](agent-management.md)**
     *에이전트를 관리하는 방법에 대해 알아봅니다.*
* 자동화된 채팅
   * **[개인 맞춤화된 대화 상자 생성 및 관리](dialogue-management.md)**
     *[!UICONTROL Dynamic Chat]을 사용하여 웹 방문자의 빠른 참여를 유도하는 대화를 만드는 방법에 대해 알아봅니다.*
   * **[대화에 PDF 임베드](document-cloud-integration.md)**
     *[!UICONTROL Dynamic Chat]에서 설계한 개인 맞춤화된 대화 내에서 PDF 문서를 공유합니다.*
   * **[스트림 디자이너에서 검색](search-in-stream-designer.md)**
     *빠른 검색을 통해 한 번의 클릭으로 스트림 디자이너의 모든 카드로 이동할 수 있습니다.*
   * **[유추한 속성 캡처](capture-inferred-attributes.md)**
     *Munchkin ID를 활용하여 Dynamic Chat을 통해 캡처된 모든 리드에 대해 유추한 속성을 식별하고 채울 수 있습니다.*
   * **[대화 미리보기](dialogue-preview.md)**
     *대화 미리보기 기능을 사용하면 게시 전 대화를 매우 쉽게 미리 볼 수 있습니다.*
   * **[정적 콘텐츠의 글로벌화](globalization-of-static-content.md)**
     *선택한 언어로 정적/시스템 콘텐츠를 개인화할 수 있습니다.*
   * **[대화 양식](conversational-forms.md)**
     *사용자가 양식을 작성한 후 즉시 웹 사이트 방문자의 참여를 유도하는 방법에 대해 알아봅니다.*
   * **[Conversations SDK](conversations-sdk.md)**
     *모든 CTA 클릭을 참여도 높은 대화로 전환하기 위해 Conversations SDK를 구현하는 방법에 대해 알아봅니다.*
   * **[재사용 가능한 플로우](reusable-flows.md)**
     *새로운 대화 흐름 카드를 사용하여 대화 설계 및 관리를 간소화합니다. 대화 스트림을 한 번 만들고 여러 대화에 재사용합니다.*
   * **[대화형 랜딩 페이지](conversational-landing-pages.md)**
     *몇 번의 클릭만으로 모든 랜딩 페이지(Marketo 랜딩 페이지 또는 사용자 정의)에 Dynamic Chat 대화 흐름을 설치하고 모든 링크나 버튼을 대화 참여를 트리거하도록 전환합니다.*
* 라이브 채팅 및 회의 예약
   * **[방문자를 라이브 에이전트에 연결](connect-visitors-to-live-agents.md)**
     *자격을 갖춘 리드를 연결하여 영업 에이전트와 즉시 참여하게 합니다.*
   * **[리드가 영업 담당자와 미팅을 예약할 수 있도록 합니다.](meeting-booking.md)**
     *회의 예약 기능을 사용하여 [!DNL target] 계정에 대해 영업 담당자와의 연결을 가속화합니다.*
* 생성형 AI 채팅
   * **[생성형 AI 기반 대화 유도](gen-ai-features.md)**
     *Dynamic Chat 생성형 AI 기능을 사용하는 방법에 대해 알아봅니다.*
* Analytics
   * **[참여도 보고](engagement-report.md)**
